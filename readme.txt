== Description ==

This plugin lets you to create a post form by using shortcode 'post_form' and insert a new post while submitting the data.

== Features ==

* Create a form using  shortcode 'post_form'
* Insert a new post upon data submission

== Installation ==

Install the plugin via WordPress Dashboard (upload from your computer), or upload the files manually to your server and activate the plugin.

== Frequently Asked Questions ==

= How to enable the plugin? =

Activate the plugin. That's it.

= How to disable the plugin? =

Just deactivate the plugin.
